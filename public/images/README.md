# Surgical Images

Please place your surgical procedure images in this directory:

1. `surgery-1.jpg` - First surgical image
2. `surgery-2.jpg` - Second surgical image  
3. `surgery-3.jpg` - Third surgical image
4. `surgery-4.jpg` - Fourth surgical image

**Image Requirements:**
- Format: JPG, PNG, or WebP
- Recommended size: 1200x800px or larger
- Aspect ratio: 4:3 works best for the gallery layout

The images will be displayed in:
- Hero section background (surgery-1.jpg)
- Surgical Gallery section (all 4 images)
